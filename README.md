## 2024 데이터 크리에이터 캠프: 패션 스타일 분류 및 추천 시스템
본 프로젝트는 2024 데이터 크리에이터 캠프 대학부 출제 문제를 수행한 결과물로, ResNet-18 아키텍처를 직접 구현하여 패션 스타일 이미지 분류와 추천 시스템을 구축하는 것을 목적으로 합니다.

# 기술적 제약 사항 (Constraints)
본 프로젝트는 공정한 성능 평가를 위해 다음과 같은 엄격한 기술적 제약 조건을 준수하여 설계되었습니다:

-모델 아키텍처: CNN 기반의 ResNet-18 아키텍처를 사용
-학습 방식: Pre-trained Weights를 사용할 수 없으며, 모든 파라미터는 **무작위 초기화(Random Initialization)**를 통해 처음부터 학습을 진행
-성능 평가지표: 모델의 성능 평가 기준은 Top-1 Accuracy를 사용

# 미션
Mission 1. 패션 스타일 이미지 분류
1-1) 데이터 통계: 주어진 이미지 파일명({W/T}{이미지ID}{시대별}{스타일별}{성별}.jpg)을 파싱하여, 이미지 ID 기준으로 성별 및 스타일별 통계표를 작성하였습니다.
1-2) 분류 모델링: ResNet-18 아키텍처를 활용하여 성별 및 스타일 단위의 클래스 분류를 수행하였으며, Training 및 Validation 데이터에 대한 정확도를 산출하였습니다.(본인 담당)

Mission 2. 패션 스타일 선호 여부 예측

2-1) 데이터 정제: 이미지 데이터와 라벨링 데이터(JSON)를 매칭하여 유효한 라벨링 데이터만을 식별하였습니다. (본인 담당)
2-2) 선호도 분석: 응답자(User ID)별로 스타일 선호 정보표를 구축하였습니다. 스타일 선호 여부는 item>survey>Q5 필드(1: 비선호, 2: 선호)를 기준으로 추출하였습니다. (본인 담당)

Mission 3. 패션 스타일 추천 시스템 구현

3-1) 이론적 분석: 협업 필터링(Collaborative Filtering)의 User-based 및 Item-based 기법을 이해하고, 장단점을 비교 분석하였습니다.
3-2) Item-based Filtering 구현: Mission 1에서 학습된 ResNet-18 모델의 중간 레이어(Intermediate Layer) 값을 추출하여 각 이미지의 특징 벡터(Feature Vector)를 생성
+ 벡터 연산을 통해 이미지 간 유사도(image2image)를 계산하고, 이를 토대로 Validation 데이터 내 응답자의 스타일 선호 여부를 예측 및 성능을 측정하였습니다.
